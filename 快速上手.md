# 🎯 快速上手 - 5分钟启动系统

## ⚡ 最快启动方式（使用 IDEA）

### 第一步：启动后端（2分钟）

1. **在 IDEA 中打开项目**
   - File → Open → 选择 `D:\project\springboot3wd7d5i4`

2. **配置数据库**
   ```sql
   -- 在 MySQL 中执行
   CREATE DATABASE springboot3wd7d5i4 CHARACTER SET utf8mb4;
   USE springboot3wd7d5i4;
   SOURCE D:/project/springboot3wd7d5i4/db/springboot3wd7d5i4.sql;
   ```

3. **修改配置文件**（如果需要）
   - 文件：`src/main/resources/application.yml`
   - 修改数据库密码：第14行 `password: 你的密码`

4. **运行主类**
   - 找到：`src/main/java/com/SpringbootSchemaApplication.java`
   - 右键 → Run 'SpringbootSchemaApplication'
   - 等待看到：`Started SpringbootSchemaApplication in X.XXX seconds`

### 第二步：启动前端（3分钟）

1. **打开终端**
   - IDEA 底部点击 Terminal
   - 或按 `Alt + F12`

2. **进入前端目录**
   ```bash
   cd src/main/resources/admin/admin
   ```

3. **安装依赖**（首次运行，需要等待）
   ```bash
   npm install
   ```
   ⏳ 这个过程可能需要 5-10 分钟，请耐心等待...

4. **启动开发服务器**
   ```bash
   npm run serve
   ```
   
5. **访问系统**
   - 浏览器打开：http://localhost:8081
   - 默认账号：admin / admin

## 📸 关于"前端页面没变"的问题

### 原因分析
如果您修改了前端代码但页面没变化，可能是：

1. **浏览器缓存** 
   - 解决：按 `Ctrl + Shift + R` 强制刷新
   - 或：打开开发者工具（F12）→ Network → Disable cache

2. **前端服务未重启**
   - 解决：停止前端（Ctrl+C），重新运行 `npm run serve`

3. **修改了错误的文件**
   - 确认修改的是：`src/main/resources/admin/admin/src/` 下的文件
   - 而不是：`dist/` 目录（这是编译后的文件）

### 💡 开发建议

**实时重载**：
- Vue 开发服务器支持热更新
- 修改 `.vue` 文件后自动刷新
- 无需手动重启

**查看修改效果**：
```bash
# 确保前端在运行
npm run serve

# 修改 src/ 下的文件
# 保存后浏览器自动刷新
```

## 🆘 紧急问题排查

### 问题 1：后端启动失败

**检查清单**：
- [ ] MySQL 是否运行？
- [ ] 数据库是否创建？
- [ ] 用户名密码是否正确？
- [ ] 端口 8080 是否被占用？

**快速修复**：
```bash
# 查看端口占用
netstat -ano | findstr :8080

# 终止进程
taskkill /F /PID <PID>
```

### 问题 2：前端白屏

**检查清单**：
- [ ] 后端 API 是否正常？
- [ ] 浏览器控制台有错误吗？（F12）
- [ ] 网络连接正常吗？

**快速修复**：
```bash
# 清除缓存重新安装
rm -rf node_modules
rm package-lock.json
npm install
npm run serve
```

### 问题 3：npm install 太慢

**解决方案**：
```bash
# 使用淘宝镜像
npm config set registry https://registry.npmmirror.com

# 重试
npm install
```
